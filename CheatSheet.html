My CheatSheet

My little CheatSheet. Shortcuts and commands I'm often using but can't remember.

GIT

git add -p Partial add. You get to chose which changes you want to add to next commit and which not.
git reset --hard Remove all uncommited changes.

Eclipse

ALT+SHIFT+A Block selection/edition mode. Select rows with SHIFT and arrow keys, end with main shortcut.
CTR+SHIFT+R Open resource.
CTR+SHIFT+T Open type.

Linux

tar -xzf file.tar.gz Unpack a tarball (*.tar.gz file).
nohup command_name & Execute command that won't get killed after exiting shell prompt.
